# 项目：企业合作对接平台 - API 接口需求文档

**版本**：V1.0

**日期**：2025-05-18

**状态**：初稿

## 1. 概述

本文档定义了“企业合作对接平台”大屏可视化界面的后端接口需求。界面主要包含核心指标统计、图表分析、AI智能交互及公共服务四大模块。

**全局说明**：

* 所有接口建议使用 `HTTPS` 协议。
* 响应格式统一为 JSON。
* 基础 URL 示例：`https://api.yourdomain.com/v1`

---

## 2. 接口详情

### 2.1 模块一：核心指标看板 (Dashboard Overview)

**描述**：用于左侧面板展示专家、企业、产品的核心统计数据。

#### 接口 1：获取平台核心概览数据

* **接口地址**: `/api/dashboard/overview`
* **请求方式**: `GET`
* **请求参数**: 无
* **响应示例**:
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "expert": {
      "total": 8540,
      "trend": 12,        // 12%
      "trend_dir": "up"   // 趋势方向
    },
    "enterprise": {
      "total": 12300,
      "trend": 8,
      "trend_dir": "up"
    },
    "product": {
      "total": "50,300+" // 字符串格式，包含特殊符号
    }
  }
}

```



---

### 2.2 模块二：数据分析图表 (Charts & Analytics)

**描述**：用于右侧面板展示行业分布、成交金额及匹配次数的图表数据。

#### 接口 2：获取统计图表数据

* **接口地址**: `/api/dashboard/charts`
* **请求方式**: `GET`
* **请求参数**:
* `range`: `string` (可选，例如 "half_year", "year")


* **响应示例**:
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "industry_pie": [
      { "name": "人工智能", "value": 35 },
      { "name": "智能制造", "value": 40 },
      { "name": "生物医药", "value": 25 }
    ],
    "amount_bar": {
      "xAxis": ["1月", "2月", "3月", "4月", "5月", "6月"],
      "series": [1400, 2000, 1700, 1750, 1100, 1200],
      "unit": "万"
    },
    "match_line": {
      "xAxis": ["1月", "2月", "3月", "4月", "5月", "6月"],
      "series": [1200, 1700, 1100, 1800, 2000, 1500],
      "unit": "次"
    }
  }
}

```



---

### 2.3 模块三：AI 智能助手 (AI Assistant)

**描述**：中间区域的核心交互模块，包含提示词推荐和对话搜索。

#### 接口 3：获取推荐提示词 (Prompt Suggestions)

* **接口地址**: `/api/ai/prompts`
* **请求方式**: `GET`
* **描述**: 加载输入框上方的快捷提问标签。
* **响应示例**:
```json
{
  "code": 200,
  "data": [
    "问题举例合作的企业",
    "输出有需求的企业",
    "帮我罗列跟人工智能相关的企业"
  ]
}

```



#### 接口 4：AI 对话与搜索匹配 (Chat & Search)

* **接口地址**: `/api/ai/chat`
* **请求方式**: `POST`
* **描述**: 处理用户输入的自然语言，返回文本回复或结构化数据列表。
* **请求参数**:
```json
{
  "query": "帮我罗列跟人工智能相关的企业",
  "session_id": "uuid-1234-5678"
}

```


* **响应示例**:
```json
{
  "code": 200,
  "data": {
    "reply_type": "list", // "text" 纯文本 | "list" 数据列表
    "answer_text": "好的，已为您找到以下与人工智能相关的企业：",
    "payload": [
      { "id": 101, "name": "深蓝科技有限公司", "category": "AI算法" },
      { "id": 102, "name": "未来视觉实验室", "category": "计算机视觉" }
    ]
  }
}

```



---

### 2.4 模块四：公共服务 (Common Services)

**描述**：顶部导航栏的基础信息。

#### 接口 5：获取天气信息

* **接口地址**: `/api/common/weather`
* **请求方式**: `GET`
* **响应示例**:
```json
{
  "code": 200,
  "data": {
    "city": "杭州",
    "condition": "中雨",
    "temperature": 27
  }
}

```

## 3. 接口汇总表

| ID | 接口名称 | Method | URI | 前端调用时机 |
| --- | --- | --- | --- | --- |
| 1 | 获取概览数据 | GET | `/api/dashboard/overview` | 页面加载 |
| 2 | 获取图表数据 | GET | `/api/dashboard/charts` | 页面加载 |
| 3 | 获取推荐提示词 | GET | `/api/ai/prompts` | 页面加载/对话结束 |
| 4 | AI 对话搜索 | POST | `/api/ai/chat` | 用户发送消息 |
| 5 | 获取天气信息 | GET | `/api/common/weather` | 页面加载 |
