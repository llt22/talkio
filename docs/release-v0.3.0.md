## What's New

### 🔑 API Key 可选
支持无需 API Key 的供应商（如 HuggingFace Spaces、Ollama 本地部署等）。添加供应商时 API Key 字段标注为可选，留空即可直接连接。

### 🗑️ 统一删除交互
全 App 删除操作统一为左滑删除：
- 供应商列表、角色卡片、MCP Server 卡片均支持左滑删除
- 红色背景 + 垃圾桶图标 + 缩放动画，交互一致
- 删除前弹出确认对话框，防止误操作

### 🧪 供应商连接流程优化
- 测试连接不再自动保存供应商，避免重复创建
- 仅在点击"保存"时才写入存储
- 新增同名供应商校验，防止重复添加

### ⬇️ 滚动到底部按钮重设计
- 位置：从屏幕中间移到消息列表底部（输入框正上方）
- 动画：弹簧淡入 + 缩放效果，点击后平滑淡出
- 样式：药丸形主题色按钮，更现代

### 🎨 视觉风格统一
- 移除全 App 不必要的外边框（border），改用背景色对比区分层级
- 涉及设置页、供应商编辑、角色编辑、工具编辑、对话气泡等
- 发现页 Tab 切换器在日间模式下清晰可见
- 连接按钮在日间模式下文字可见（修复白色文字 + 浅色背景问题）
