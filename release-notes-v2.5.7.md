## What's New in v2.5.7

### ✨ New Features
- 对话置顶功能：桌面端右键/三点菜单、移动端更多菜单支持置顶/取消置顶，置顶对话在列表中优先显示并带图钉标识
- 供应商列表新增一键刷新所有供应商模型按钮，刷新后每个供应商状态点实时变色反馈（灰→绿/红）+ toast 汇总通知

### 🐛 Bug Fixes
- 修复群聊重试消息排序问题：重试的消息现在正确排在末尾，而非跑到第一个发言者后面
- 修复 MentionTextarea 中同模型参与者 @提及无法区分的问题，使用唯一标签（#1 #2 或角色名）区分
- 增强 AbortSignal.timeout polyfill，修复移动端 WebView 静默失败问题
